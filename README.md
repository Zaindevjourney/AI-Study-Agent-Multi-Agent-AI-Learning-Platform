# AI Study Agent | Multi-Agent AI Learning Platform

## 📖 Overview

**AI Study Agent** is a full-stack AI-powered learning platform built with **Python, FastAPI, Large Language Models (LLMs), Retrieval-Augmented Generation (RAG), and a Multi-Agent AI Architecture**. The platform is designed to provide students with an intelligent, personalized, and interactive learning experience by combining multiple AI agents into a single unified system.

Students can upload PDF study materials, ask questions to an AI Tutor, generate study notes, quizzes, and flashcards, create personalized study plans, and track learning progress through a modern responsive dashboard. The system leverages **Retrieval-Augmented Generation (RAG)** and **semantic search** to answer questions using uploaded documents while maintaining long-term learning context.

At the core of the platform is an **Orchestrator Agent** that intelligently routes user requests to specialized AI agents, enabling a modular, scalable, and extensible architecture.
